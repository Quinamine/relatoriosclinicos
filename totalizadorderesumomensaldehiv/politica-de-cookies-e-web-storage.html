Política de Cookies e Web Storage

Cookies é um recurso de armazenamento local (no dispositivo do usuário) de informações por meio de pequenos arquivos de texto que a maioria dos sites utiliza para guardar predefinições. Eles são de suma importância pois são como anotações que os sites fazem sobre o comportamento de navegação de cada visitante e posteriormente recorrem-nas para recordar-se das acções e preferências do visitante a fim de oferecê-lo a melhor experiência possível. É graças maioritariamente aos cookies que um site ou aplicativo mantém um usuário logado por exemplo, sem a necessidade desse digitar a senha toda vez que abre o aplicativo.

Para salvar os dados da ficha no teu navegador, o Totalizador de Resumo Mensal de HIV/SIDA faz uso de uma tecnologia semelhante designada Web Storage.

Existem 2 tipos de Web Storage:

  • Session Storage (Armazenamento de Sessão) – elimina os dados gravados quando a janela da página é fechada.
  • Local Storage (Armazenamento Local) – mantém os dados salvos por um período determinado ou até que os excluas.

O Totalizador utiliza o primeiro para guardar o teu consentimento à utilização de "cookies" e o último pelos três motivos abaixo:

Fazer um backup local
Fazer uma cópia de segurança dos dados da ficha no teu navegador até que os excluas. Se forem preenchidos, os dados que são salvos incluem:
  • Os dados da ficha propriamente dita.
  • Dados adicionais como:
    • Província;
    • Distrito;
    • Nome da Unidade Sanitária;
    • NID da Unidade Sanitária;
    • O teu nome (como Profissional responsável pelo preenchimento);
    • Data de elaboração.
Durabilidade dos dados
Manter salvos os dados previamente inseridos no acto de preenchimento da ficha por tempo suficiente para serem aproveitáveis. Isso é importante, pois possibilita-te interromper sem o risco de voltar à estaca zero, a elaboração do resumo para continuar num outro momento oportuno, sobretudo se ainda te faltar dados de alguns indicadores para preencher nas respectivas células da ficha.
Salvar preferências
Por exemplo, quando marcas a opção “Destacar as células automáticas", essa escolha é salva no Local Storage para o Totalizador manter nas tuas próximas visitas à plataforma.

Destino e tratamento dos dados

O Totalizador de Resumo Mensal de HIV/SIDA foi desenvolvido apenas com recurso a tecnologias client-side (que funcionam no do lado do cliente ou usuário, isto é, no navegador) e não existe nenhum banco de dados com que está conectado que salva os dados da ficha. Todas as alterações são salvas localmente (no navegador). Em suma, o serviço não tem uma cópia remota dos dados. O armazenamento local é para fins principalmente de memorização e com isso melhorar a tua experiência com a plataforma.

Gerenciamento de cookies e Web Storage

Por padrão os navegadores estão habilitados para permitir o Armazenamento Local e de Cookies, porém é possível desabilitá-los nas configurações do teu navegador (embora não seja recomendável). Como alternativa podes optar por apagar os cookies armazenados em vez de desabilitar a funcionalidade. No menu do Totalizador, existe a opção “esvaziar ficha”, que te permite apagar todos os dados da ficha salvos pelo Totalizador no Local Storage. Em paralelo, sempre que preferires, poderás excluir os dados salvos através das configurações do teu navegador. Para mais detalhes sobre o gerenciamento de cookies:

  • Veja aqui se estiver usando o Microsoft Edge;
  • Veja aqui se estiver usando o Google Chrome;
  • Veja aqui se estiver usando o Modzilla Firefox;
  • Veja aqui se estiver usando o Safari.
