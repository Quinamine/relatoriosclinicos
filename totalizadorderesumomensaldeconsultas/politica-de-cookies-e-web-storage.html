Política de Cookies e Web Storage

O que são cookies?

Cookies são pequenos arquivos de texto criados por algorítimos computacionais e armazenados no seu navegador para guardar algumas informações. A maioria dos sites utiliza os cookies para armazenar predefinições de cada visitante no seu respectivo navegador e posteriormente recorrem-nos para recordar-se das preferências do visitante. Por exemplo, quando um visitante activa o modo escuro, tal preferência pode ser armazenada no seu dispositivo em forma de cookie para manter nas suas próximas visitas ao site.

O Totalizador de Resumo Mensal de Consultas Externas faz uso de uma tecnologia paralela designada Web Storage (um espaço disponível nos navegadores também para guardar algumas informações). Basicamente, existem 2 tipos de Web Storage ou Armazenamento:

  • Armazenamento de Sessão ou Session Storage – que elimina os dados gravados quando a janela do navegador é fechada.
  • Armazenamento Local ou Local Storage – que mantém os dados salvos por um período determinado ou até que o usuário os exclua.

O Totalizador utiliza a Local Storage por três (3) motivos, a destacar:

Fazer um backup local
Fazer uma cópia de segurança dos dados da ficha no seu navegador até que você os exclua. Se forem inseridos, os dados que são salvos incluem:
  • Nome da US;
  • Mês;
  • Nome do Responsável pelo preenchimento;
  • Categoria;
  • Data da elaboração;
  • E os dados da ficha propriamente dita.
Durabilidade dos dados
Manter salvos os dados previamente inseridos no acto de preenchimento da ficha por tempo suficiente para serem aproveitáveis. Isso é importante, pois o possibilita interromper sem o risco de voltar à estaca zero, a elaboração do resumo para continuar num outro momento oportuno, sobretudo se ainda lhe faltar dados de alguns indicadores para preencher nas respectivas células da ficha.
Salvar as suas preferências
Por exemplo, quando você marca a opção “Destacar as células automáticas", essa escolha é salva no Local Storage para manter nas suas próximas visitas à plataforma.

Destino e tratamento dos dados

O Totalizador de Resumo Mensal de Consultas Externas foi desenvolvido apenas com recurso a tecnologias client-side (que funcionam no navegador, lado do cliente ou usuário) e não existe nenhum banco de dados com que está conectado que salva os dados da ficha. Todas as alterações são salvas localmente (no seu navegador). Em suma, o serviço não tem uma cópia remota dos dados que você insere. O armazenamento local dos dados é para fins principalmente de memorização e com isso melhorar a sua experiência com a plataforma.

Gerenciamento dos dados

Por padrão os navegadores estão habilitados para permitir o Armazenamento Local e de Cookies, mas você pode desabilitá-los nas configurações do seu navegador (embora não seja recomendável). Como alternativa você pode optar por apagar os cookies armazenados em vez de desabilitar a funcionalidade. No menu do Totalizador, existe a opção “esvaziar ficha”, que o permite apagar todos os dados da ficha salvos pelo Totalizador no Local Storage. Em paralelo, sempre que preferir, poderá excluir os dados salvos através das configurações do seu navegador. Aprenda mais sobre o gerenciamento de cookies, nos seguintes endereços:

  • Veja aqui se estiver usando o Google Chrome;

  • Veja aqui se estiver usando o Safari.
