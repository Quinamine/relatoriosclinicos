Política de cookies e Web Storage

Cookies é um recurso de armazenamento local (no navegador do usuário) de dados em forma de texto que a maioria dos sites utiliza para guardar predefinições. Eles são de suma importância pois são como anotações que os sites fazem sobre o comportamento de navegação de cada usuário e posteriormente recorrem-nas para recordar-se das acções e preferências do usuário a fim de oferecê-lo a melhor experiência possível. É graças maioritariamente aos cookies que um site ou aplicativo mantém um usuário logado ou o modo escuro activo por exemplo.

Para salvar os dados da ficha no navegador do usúario, o Totalizador de Resumo Mensal de HIV/SIDA utiliza uma tecnologia semelhante designada Web Storage.

Existem 2 tipos de Web Storage:

  • Session Storage (Armazenamento de Sessão) – elimina os dados gravados quando a janela da página é fechada.
  • Local Storage (Armazenamento Local) – mantém os dados salvos por um período determinado ou até que o usuário os exclua.

O Totalizador utiliza o primeiro para guardar o consentimento do usuário à utilização de "cookies" e o último pelos três motivos abaixo:

Fazer um backup local
Fazer uma cópia de segurança dos dados da ficha no navegador do usuário até que este os exclua. Se forem preenchidos, os dados que são salvos incluem:
  • Os dados da ficha propriamente dita.
  • Dados adicionais como:
    • Província;
    • Distrito;
    • Nome da Unidade Sanitária;
    • NID da Unidade Sanitária;
    • Nome do Profissional;
    • Data de elaboração.
Durabilidade dos dados
Memorizar e mais tarde manter os dados do usuário previamente inseridos no acto de preenchimento da ficha. Isso é importante, pois possibilita ao Profissional interromper sem o risco de voltar à estaca zero, a elaboração do resumo para continuar num outro momento oportuno, sobretudo se ainda lhe faltar dados de alguns indicadores para preencher nas respectivas células da ficha.
Salvar preferências
Por exemplo, quando o usuário marca a opção “Destacar as células automáticas", essa escolha é salva no Local Storage para manter nas suas próximas visitas à plataforma.

Gerenciamento de cookies e Web Storage

Por padrão os navegadores estão habilitados para permitir o Armazenamento Local e de Cookies, porém o usuário pode desabilitá-los nas configurações do seu navegador (embora não seja recomendável). Como alternativa o usuário pode optar por apagar os cookies armazenados ao invés de desabilitar a funcionalidade. No menu do Totalizador, existe a opção “esvaziar ficha”, que o permite apagar todos os seus dados da ficha salvos pelo Totalizador no Local Storage. Em paralelo, sempre que o usuário quiser poderá excluir os seus dados salvos através das configurações do seu navegador. Para mais detalhes sobre o gerenciamento de cookies:

  • Veja aqui se estiver usando o Microsoft Edge;
  • Veja aqui se estiver usando o Google Chrome;
  • Veja aqui se estiver usando o Modzilla Firefox;
  • Veja aqui se estiver usando o Safari.
